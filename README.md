# Python Console Games

This repository contains simple console-based games built using Python.
The projects focus on programming logic, condition handling, and structured
code rather than graphics or UI.

## Games Included

### 1. Hand Cricket
A console-based version of the Hand Cricket game.

**Features**
- Toss system
- Batting and bowling logic
- Target-based second innings
- Score comparison and result declaration

**Concepts Used**
- Loops and conditional statements
- Functions and modular code
- Random number generation

### 2. Rock Paper Scissors
A classic Rock–Paper–Scissors game played against the computer.

**Features**
- Best-of-five series
- Score tracking
- Input validation

**Concepts Used**
- Functions
- Loop control
- Decision-making logic

## How to Run
Make sure Python is installed.

```bash
python hand_cricket.py
python rock_paper_scissors.py
